# Project 1 - Robotic Arm Kinematics and Path Planning

## Objective
To calculate robotic arm joint angles using inverse kinematics and generate a movement path between coordinates.

## Software Used
- Python
- PyCharm
- Git
- GitHub
- Matplotlib

## Project Files

### ik_solver.py
Calculates the robotic arm joint angle for a target coordinate.

### path_planner.py
Generates a path consisting of waypoints for robotic arm movement.

### visualization.py
Visualizes the planned path using a graph.

## Output

### Inverse Kinematics Output
- Calculates joint angle for target coordinates.

### Path Planning Output
- Generates a sequence of movement points.

### Visualization Output
- Displays the robot path graphically.

## Conclusion
The project demonstrates basic robotic arm kinematics and path planning concepts using Python.
